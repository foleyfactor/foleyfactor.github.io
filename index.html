My Projects

Here are the things that I've made and am making.

timber

1st place at Waterloo Codefest!

timber is an app that connects citizens through city-owned trees. Users can "treetup" (meet-up at a tree), vote on their favourite trees, and report dead or dangerous trees to the city.

Built with: HTML, CSS, Javascript/JQuery, Intel XDK, Firebase, Geofire

Check it out!

Sandwich-o-matic

Best Hardware Hack, Finalist at ECHacks!

Sandwich-o-matic is an automated IOT voice-controlled sandwich maker. Users simply say "make me a sandwich with [toppings] and the machine toasts the bread, dispenses the toppings, and creates the requested sandwich.

Built with: Arduino/Photon, Hardware Components

Check it out!

Intercode

Top 100 at hack.summit() 2016!

Intercode is a web application that allows users to learn how to program in Python, based on their interests. Users tell the application what they're interested in and Intercode tailors the lessons to these interests.

Built with: HTML, CSS, Javascript/JQuery, Firebase

Check it out!

CheqOut

CheqOut is an Android app that relieves the pains of waiting in lines. Scan the items as you put them in your cart, pay with your card, and show the receipt to an attendant as you walk out the door.

Built with: Android, Firebase

Check it out!

LEAP

LEAP is an Android application that allows users to use their phones in a similar way to leap motion. The app uses the phone's front camera to track the user's hands and then sends commands to a computer, based on their gestures.

Built with: Android, OpenCV, Firebase

Check it out!

Plan Me

Plan Me is a more intelligent planning platform. Users input various tasks that they must complete for the week and Plan Me schedules all of the tasks, according to the user's personality. If there isn't enough time, the higher priority events will be scheduled instead of the low priority ones.

Built with: HTML/CSS, JavaScript (React/Redux), Firebase

Check it out!
